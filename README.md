# Task 3 - Basic Vulnerability Scan

## Objective
To perform a basic vulnerability scan on my local computer using Nessus Essentials.

## Tool Used
- Nessus Essentials

## Target
- Local PC (192.168.1.5)

## Procedure
1. Installed Nessus Essentials.
2. Configured a Basic Network Scan.
3. Scanned the local machine using the local IP address.
4. Reviewed the identified vulnerabilities.
5. Collected screenshots and scan details.

## Findings
- Scan Status: Completed
- Total Vulnerabilities: 27
- Authentication: Failed (Unauthenticated Scan)

## Mitigations
- Review identified vulnerabilities.
- Apply Windows updates.
- Keep installed software up to date.
- Disable unnecessary services if required.

## Files Included
- Screenshots
- Vulnerability details (.txt)
- README.md

## Outcome
Successfully performed a basic vulnerability assessment and understood how vulnerability scanners identify security issues.
